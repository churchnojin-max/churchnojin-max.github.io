# 노진교회 컴퓨터 설치 안내 (30분)

받으신 압축 파일은 **백업용**입니다. 압축을 풀어 보관만 하시고,
실제 작업은 아래 순서대로 하면 인터넷(GitHub)에서 최신 자료를 새로 받아 시작합니다.

---

## 1단계 — 압축 파일 보관 (1분)

- 압축을 풀어 `문서` 폴더 등에 **"노진교회홈페이지-백업"** 이름으로 저장해 두세요.
- 이 폴더는 만일을 대비한 보관용입니다. 여기서 파일을 고치지 마세요.

## 2단계 — Claude(클로드) 설치 (5분)

1. 인터넷에서 **claude.com/download** 접속 → Windows용 내려받아 설치
2. 클로드 실행 → 로그인 (교회에서 쓸 계정으로 가입하거나 안내받은 계정 사용)

## 3단계 — 홈페이지 폴더 새로 받기 (10분)

1. 클로드를 열고 아래 문장을 **그대로 복사해 붙여넣으세요**:

> 노진교회 홈페이지를 관리하려고 해. https://github.com/churchnojin-max/churchnojin-max.github.io.git 저장소를 바탕화면에 "노진교회홈페이지" 폴더로 받아줘. Git이 설치되어 있지 않으면 설치부터 안내해줘.

2. 클로드가 화면에 나오는 대로 안내합니다. 중간에 허락을 물으면 내용을 읽고 "예"를 누르세요.

## 4단계 — GitHub 로그인 연결 (5분)

수정한 내용을 실제 홈페이지에 올리려면 GitHub 로그인이 한 번 필요합니다.
클로드에게 이렇게 말하세요:

> 이 폴더에서 GitHub에 올릴 수 있는지 확인해줘. 로그인이 필요하면 churchnojin-max 계정으로 로그인하는 방법을 안내해줘.

- **churchnojin-max 계정의 비밀번호**는 인수인계 받은 안전한 곳에 보관된 것을 사용하세요.

## 5단계 — 동작 확인 (5분)

클로드에서 "노진교회홈페이지" 폴더를 연 상태로:

> 홈페이지 하단 저작권 연도가 잘 나오는지 확인만 해줘. 그리고 지금 이 폴더에서 수정하면 실제 홈페이지에 반영되는지 시험해줘.

문제없다는 답을 받으면 설치 끝입니다.

## 6단계 — 홈페이지 앱 설치 (2분)

1. Edge 또는 Chrome에서 `churchnojin-max.github.io` 접속
2. 주소창 오른쪽의 **"앱 설치"** 아이콘 클릭 → 설치
3. 바탕화면에 생긴 "노진교회" 아이콘으로 홈페이지를 앱처럼 사용

## 이후 사용법

- 자세한 사용법은 새로 받은 폴더 안의 **`목사님-사용설명서.md`** 파일에 있습니다.
  (클로드에게 "사용설명서를 화면에 보여줘"라고 해도 됩니다)
- 평소에는 클로드를 열고 한국말로 부탁만 하면 됩니다. 예:
  - "주일예배 시간을 오전 11시로 바꿔줘"
  - "방금 바꾼 거 되돌려줘"
- 공지·주보·사진·교적·재정은 홈페이지에 로그인해서 관리자 메뉴에서 올립니다.
