# 노진교회 홈페이지 — Claude 작업 지침

## 이 프로젝트는

노진교회(경기도 화성시 장안면, 예장합동, 담임 손병민 목사) 홈페이지입니다.
- 정적 HTML/CSS/JS 사이트. 빌드 과정 없음.
- 배포: GitHub Pages. `main` 브랜치에 푸시하면 1~2분 뒤 https://churchnojin-max.github.io/ 에 자동 반영됩니다.
- 로그인·게시판·교적·재정 데이터는 Supabase(프로젝트 `vwuzmklacdwiqyqjrxyt`)에 있습니다.

## 사용자에 대해 (매우 중요)

이 컴퓨터의 주 사용자는 **컴퓨터에 익숙하지 않은 목사님**입니다. 반드시 지키세요:

1. **쉬운 한국어로만** 대화하세요. 전문용어(커밋, 푸시, 브랜치, 배포, 렌더링 등)는 쓰지 말고, 쓰게 되면 반드시 풀어서 설명하세요. 예: "푸시했습니다" ❌ → "인터넷 홈페이지에 올렸습니다. 1~2분 뒤에 반영됩니다" ⭕
2. **짧고 친절하게.** 선택지를 여러 개 나열하며 묻지 말고, 가장 좋은 방법 하나로 진행한 뒤 결과를 알려드리세요.
3. 작업이 끝나면 **무엇이 어떻게 바뀌었는지 한두 문장으로** 설명하고, "홈페이지에서 새로고침하면 보입니다"처럼 확인 방법을 알려드리세요.
4. 목사님의 요청이 모호해도 되물음을 최소화하고, 상식적인 방향으로 진행한 뒤 "이렇게 해 두었습니다. 다르게 원하시면 말씀해 주세요"라고 하세요.

## 작업 규칙

- **수정을 마치면 항상 자동으로 저장(커밋)하고 `main`에 푸시하세요.** 목사님은 "인터넷에 올려줘"라고 따로 말하지 못할 수 있습니다. 커밋 메시지는 한국어로 간단히.
- **되돌리기 요청**("아까 거 취소해줘")은 `git revert`로 처리하세요. `git reset --hard`, 강제 푸시(`--force`)는 절대 쓰지 마세요.
- **`js/config.js`의 열쇠 값(Supabase 주소·키)은 절대 바꾸거나 지우지 마세요.** 이 값이 지워지면 로그인·게시판·교적·재정이 전부 멈춥니다.
- **교인 개인정보(이름·연락처·헌금 내역 등)를 코드나 파일에 절대 저장하지 마세요.** 그런 데이터는 Supabase에만 있습니다.
- 파일을 크게 지우는 작업 전에는 쉬운 말로 한 번 확인받으세요.

## 자주 고치는 파일 지도

| 목사님 요청 | 고칠 곳 |
|---|---|
| 교회 이름·주소·전화·예배시간 문구·헌금 계좌·섬기는 분들 | `js/church.js` (사이트 전체 자동 반영) |
| 첫 화면(대문) 문구·구성 | `index.html` |
| 처음 오신 분·예배 안내·오시는 길 | `welcome.html` |
| 교회 소개·연혁 | `story.html` |
| 이번 주 말씀·신앙고백 | `word.html` |
| 선교·연합사역 | `world.html` |
| 교회 정관 | `bylaws.html` |
| 로고·아이콘 그림 | `images/` (파일명 유지하고 내용만 교체) |
| 사이트 색·글꼴 | `css/` |

**공지·주보·사진첩·기도제목·교적·재정 입력은 코드가 아니라 홈페이지 관리자 화면에서 합니다.**
목사님이 이런 걸 요청하면 홈페이지에 로그인해서 하는 방법을 차근차근 안내하거나, 브라우저 도구로 직접 도와드리세요.

## 미리보기

배포 전에 확인이 필요하면:
```
python -m http.server 8000
```
후 `http://localhost:8000` 접속. (간단한 수정은 바로 올려도 됩니다.)
